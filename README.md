# Career Seekers

**Career Seekers** — платформа для обслуживания чемпионата "Искатели Профессий", проводящегося в г.Санкт-Петербург.
Проект представляет собой набор микросервисов на Kotlin + Spring Boot, реализующих управление участниками чемпионата (
родители и дети), а также предоставляет ряд функций для организаторов чемпионата (эксперты компетенций, кураторы
площадок проведения мероприятий, организаторы чемпионата).

## Архитектура / Микросервисы:

**Для реализации проекта была выбрана микросервисная архитектура, так как с самого начала было понятно что придется
реализовывать большое количество разнообразного функционала. Основной способ связи между микросервисами — gRPC и GraphQL
запросы (в процессе разработки проекта была сложность в определении конечного варианта связи микросервисов - это
послужило причиной применения сразу двух технологий. В дальнейшем планируется полный переход на rpc в контексте
межсервисной связи), для задач, не требующих синхронного выполнения (задачи на отправку писем, трекинг событий,
вызывающих обновление данных на клиенте и т.п.) используется брокер очередей Apache Kafka.**

**На момент начала 2026 года существует 5 основных микросервисов, краткое описание которых представлено ниже.**

- **user-service**: Управление пользователями (родители и дети, наставники, эксперты, кураторы и организаторы),
  аутентификация пользователей, управление документами пользователей


- **events-service**: Управление площадками проведения мероприятий, компетенциями, записями на компетенции, а также
  событиями, связанными с проводимыми соревнованиями (отборочные этапы, образовательные этапы и различного рода
  мастер-классы, финальные этапы соревнований)


- **mail-service**: Сервис рассылки уведомлений


- **file-service**: Хранение/обработка документов пользователей и площадок проведения


- **api-entrypoint**: Единая точка входа для запросов к API
